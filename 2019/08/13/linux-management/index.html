光盘管理
命令
挂载光盘 mount /dev/cdrom /media
查看挂载情况 df / df -h / df -Th
卸载光盘 umount /media
开机挂载 首先进入：vi /etc/fstab ; 添加 : /dev/cdrom /media iso9660 defaults 0 0
开机挂在测试 mount -a
RPM包管理
RPM是什么：redhat package management英文缩写，只适于于Redhat和Centos系统
安装RPM包： rpm -ivh installing software
查看RPM包： rpm -qa software
查看RPM包安装的文件: rpm -ql software
反查文件是否由RPM 安装： rpm -qf path
删除RPM安装的程序： rpm -e software
