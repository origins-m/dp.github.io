vi编辑器的概述
  1. Vi是Unix及Linux系统下标准的编辑器，由美国加州大学伯克利分校开发。学会它后，您将在Linux的世界里畅行无阻。基本上vi可以分为三种状态，分别是命令模式、插入模式，和底行模式。
vi编辑器的模式
  1. 命令模式 command
  2. 输入模式 insert
  3. 末行模式 :末行命令
  4. 视图模式 visual
vi编辑器-输入模式
字符 描述
a 在当前字符的后面输入
s 删除当前字符并输入
i 在当前字符的前面输入
o 当前字符所在行下方输入
A 当前行后面输入
S 删除当前行并输入
I 当前行前面输入
O 当前字符所在行上一行输入
vi编辑器-命令模式
字符 描述
H 控制方向—左
J 控制方向—下
K 控制方向—上
L 控制方向—右
x 删除当前字符,3x删除三个字符
d^ / .d$ 当前字符删除到^行首&行尾
dd / dw 删除一行 / 删除一个单词
G / nG /dG / d1G 最后一行 / 第 n 行 / 从当前行删除到最后一行 / 从当前行删除到第一行
yy / nyy 复制当前行 / 复制n行
字符 描述
p / np / P 粘贴到下一行 / 重复粘贴np次 / 粘贴上一行
u 撤销
ctrl+r 恢复
/word 查找word单词，查找多个n键
q / q! 不保存退出 / 强制不保存退出
x / wq 保存退出,与 wq一致
w 保存不退出
%s/hello/word/g 把一篇文章中的所有hello全部替换成world,%第一行到最后的意思
1,10s/hello/word/g 把第1行到第10行的hello替换成world
Gz、Bz2和Zip压缩包操作

Gz压缩包管理

  1. 制作gz压缩包: | tar czf mydir1.tar.gz mydir1
  2. gz压缩包解压: | tar xzf mydir1.tar.gz
  3. 查看gz压缩包: | tar tf mydir1.tar.gz

Bz2压缩包管理

  1. 制作bz2压缩包: | tar cjf mydir1.tar.bz2 mydir1

  2. bz2压缩包解压: | tar xjf mydir1.tar.bz2

  3. 查看bz2压缩包: | tar tf mydir1.tar.bz2

Zip压缩包管理

  1. 制作zip压缩包: | zip -r mydir1.zip mydir1

  2. zip压缩包解压: | unzip mydir1.zip

  3. 查看zip压缩包: | unzip -l mydir1.zip
