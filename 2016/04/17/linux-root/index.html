Root 用户破解流程
  1. e ; 进入 VMware 页面 按 e 进入编辑模式。
  2. rd.break ; 在quiet后加入 rd.break。
  3. ctrl + x; 按 ctrl + x 进入 swtch_root 模式。
  4. mout -o remount,rw /sysroot ; 重新挂载分区。
  5. chroot /sysroot ; 切换根分区。
  6. passwd root ; 修改root密码，必须是8位以上复杂密码。
  7. touch /.autorelabel ; selinux 在重启后更新label。
  8. exit ； 退出。
  9. reboot ； 重启
Selinux、Firewalld和Iptables基础设置

Selinux防火墙:

  1. 查看 sestatus

  2. 关闭 vi /etc/selinux/config 修改 SELINUX=disabled

  3. 重启 init 6

firewalld防火墙:

  1. 查看 systemctl status firewalld

  2. 关闭 systemctl stop firewalld

  3. 开机关闭 systemctl disabled firewalld

iptables防火墙:

  1. 查看规则 iptables -L -n

  2. 清空规则 iptables -F

  3. 保存规则 service iptables save

Linux 系统目录结构

/ ：根目录， 包含所有文件目录，相当于中央系统

/bin ：普通用户可以使用命令的存放目录

/sbin ：超级用户可以使用命令的存放目录

/boot ：引导 程序、内核等存放的目录

/dev ： 设备文件存储目录

/etc ：全局配置文件存放目录

/home ：家目录，普通用户存放文件

/lib ：根目录下所有程序的共享目录（32位）

/lib64：根目录下所有程序的共享目录（64位）

/lost+found：恢复文件存放的位置

/media：挂载媒体设备目录

/mnt：临时挂载文件目录

/opt：程序存放目录

/proc：特殊文件目录

/root ：超级用户

/srv ： 服务创建目录

/sys ： 硬件设备的驱动程序信息

/var ： 存放内容经常变化目录

/tmp ： 临时文件目录
