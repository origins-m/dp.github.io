SSH 远程密钥操作
发布于：2020-04-24 16:34:58 标签：/ linux / 访问：

  1. Windows 生成密钥对。

    ssh-keygen -t rsa

  2. 进入 .ssh 目录

    cd C:Users/.ssh

  3. 将公钥拷贝Linux /root/.ssh 下

    spc id_rsa.pub root@192.168.2.103:/root/.ssh

  4. 在Linux上吧公钥改名

    mv id_rsa.pub authorized_keys

  5. 客户端无口令操作-命令操作

    ssh root@192.168.2.103

  6. 客户端无口令操作-文件传输

    scp index.py root@192.168.2.103:/root/
